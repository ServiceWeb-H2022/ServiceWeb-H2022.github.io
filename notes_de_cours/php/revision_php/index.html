    Révision PHP

    Syntaxe

    • Un script PHP doit être débuter par <?php et ce terminé par ?>
    • Chaque ligne de code se termine par un point-virgule.
    • Les noms de variable débutent par le symbole $

    Exécuter un script en ligne de commande

    Pour lancer un script depuis la ligne de commande 

    # Si le chemin vers l'interpreteur PHP est dans les variables d'environnement
> php monScript.php
# Sinon il faut spécifier le chemin en entier
> 'C:\Program Files\Ampps\php-7.3\php.exe' monScript.php

    Opérateur de coalescence des nuls ( ?? )

    ?? est un opérateur logique qui retourne la première valeur qui existe et qui est non null. Il est possible d'enchaîner plusieurs opérateurs de coalescence à la suite.

    <?php
// Si $_GET['utilisateur'] n'est pas défini ou vaut null, $utilisateur vaudra 'aucun' 
$utilisateur = $_GET['utilisateur'] ?? 'aucun';

// On peut enchaîner plusieurs ?? comme ceci
$utilisateur = $_GET['utilisateur'] ?? $_POST['utilisateur'] ?? 'aucun';
?>

    Tableau

    Déclaration d'un tableau

    Pour déclarer un tableau, on peut utiliser la fonction array() ou en notation courte utiliser []

    <?php
// Déclaration d'un tableau avec la fonction array()
$monTableau = array("item1", "item2", "item3");

// Déclaration d'un tableau avec la notation courte
$monTableau = ["item1", "item2", "item3"];
?>

    Chaque élément du tableau est associé à une clé (son index dans le tableau). Si la clé n'est pas spécifié, comme dans l'exemple plus haut, elle sera autogénéré de façon incrémentiel (0, 1, 2, etc...). Il est aussi possible de définir la clé de chaque item.

    <?php
$monTableau = array(
    1 => "item1",
    2 => "item2",
    3 => "item3",
);

$monTableau = [
    "fruits" => ["banane", "pomme", "raisin"],
    "legumes" => ["carotte", "patate", "navet"],
];
?>

    Accès aux éléments d'un tableau

    On accède à la valeur d'un élément du tableau par sa clé

    <?php
$monTableau = ["item1", "item2", "item3"];
echo $monTableau[0]; // "item1"

$monTableau = [
    "fruits" => ["banane", "pomme", "raisin"],
    "legumes" => ["carotte", "patate", "navet"],
];
echo $monTableau["legumes"][1]; // "patate"
?>

    Ajouter/modifier/supprimer un élément

    <?php
$monTableau = ["item1", "item2", "item3"];

// Pour ajouter un élément au tableau, on donne ou non une clé libre 
$monTableau[] = "item4";
$monTableau[4] = "item5";

// Pour modifier un élément, on doit donner la clé de l'élément à modifier
$monTableau[4] = "item5 modifié"

// On supprime un item avec la fonction unset en spécifiant la clé de l'élément à supprimer
unset($monTableau[4]);
// Attention, si aucune clé n'est donnée, c'est tout le tableau qui sera supprimé
unset($monTableau);
?>

    Boucle sur un tableau

    Pour boucler sur les éléments d'un tableau, on utilise foreach

    <?php
$monTableau = [
    "prenom" => "Mathieu",
    "nom" => "Frechette",
];

foreach($monTableau as $item) {
    echo "$item\n";
}
/*
Mathieu
Frechette
*/

// On peut aussi récupérer la clé de chaque élément
foreach($monTableau as $cle => $valeur) {
    echo "$cle = $valeur\n";
}
/*
prenom = Mathieu
nom = frechette
*/
?>

    Déballer un tableau

    En préfixant le tableau par trois points ( ...$monTableau), le tableau sera "déballé" lors de se définition, c'est à dire qu'on aura accès à tous les éléments du tableau. (Disponible depuis PHP 7.4.0)

    <?php
$monTableau = ["item1", "item2", "item3"];

// Copier le tableau
$tableau2 = [...$monTableau]; // ["item1", "item2", "item3"]
// Ajouter des éléments au tableau;
$tableau3 = ["item0", ...$monTableau, "item4"]; 
// ["item0, "item1", "item2", "item3", "item4"]

// Fusionner des tableaux
$tableau4 = [...$tableau2, ...$tableau3]; 
// ["item1", "item2", "item3", "item0, "item1", "item2", "item3", "item4"]

// On peut aussi l'utiliser avec une fonction qui retourne un tableau
function getTableau() {
    return ["item5", "item6"];
}
$tableau5 = [...getTableau(), "item7"]; //["item5", "item6", "item7"] 
?>

    Array_filter()

    La fonction array_filter() permet de filter les éléments d'un tableau par une fonction de Callback. Array_filter() prends en paramètre le tableau sur lequelle effectuer le traitement la fonction de Callback. On peut appeler une fonction existante ou bien directement la déclarer à l'intérieur de array_map().

    <?php
// Soit le tableau de nombre suivant
$nombres = [1, 2, 3, 4, 5];
// On veut créer un nouveau tableau avec uniquement les nombres impairs
$nombresImpair = array_filter($nombres,
    function ($nombre) {
        return $nombre % 2 === 1;
    }
);
// [1, 3, 5]
?>

    Array_map()

    La fonction array_map() permet d'appliquer une fonction sur chaque élément d'un tableau et de retourner un nouveau tableau du résultat. Array_map() prends en paramètre la fonction de Callback et le tableau sur lequelle effectuer le traitement. On peut appeler une fonction existante ou bien directement la déclarer à l'intérieur de array_map().

    <?php
    // Soit le tableau de nombre suivant
    $nombres = [10,12,14,13];
    // On veut créer un tableau avec de ces nombres doublés
    $nombresDouble = array_map(function ($nombre) {
        return $nombre * 2;
    }, $nombres);
    // [20, 22, 28, 26]
?>

    JSON en PHP

    Pour encoder une valeur en format JSON, on utilisera la fonction json_encode().

    <?php
$monTableau = [
    "prenom" => "Mathieu",
    "nom" => "Frechette",
];
echo json_encode($monTableau);
/* 
{
    "prenom":"Mathieu",
    "nom":"Frechette"
}
*/

$monTableauPlusComplexe = [
    "Resultats" => [
        [
            "prenom" => "Mathieu",
            "nom" => "Frechette",
        ],
        [
            "prenom" => "Anthony",
            "nom" => "Frechette",
        ],
    ]
];
echo json_encode($monTableauPlusComplexe);
/* 
{
    "Resultats":[
        {
            "prenom":"Mathieu",
            "nom":"Frechette"
        },
        {
            "prenom":"Anthony",
            "nom":"Frechette"
        }
    ]
}
*/
?>

    À l'inverse, pour convertir un objet JSON en un objet ou un tableau associatif (avec des clés définies) on utilisera la fonction json_decode(). Elle prend en paramêtre l'objet Json à convertir et optionnellement un valeur boolean qui indique si on veut que soit crée un tableau associatif ou non. Par défaut ce paramêtre prend la valeur false, ce qui crée un objet.

    <?php
$jsonData = '{"prenom":"Mathieu","nom":"Frechette"}';

$objetUsager = json_decode($jsonData);
echo $objetUsager->prenom; // Mathieu

$tableauUsager = json_decode($jsonData, true);
echo $tableauUsager["prenom"]; // Mathieu
?>

    Effectuer un appel GET à un service web

    On peut utiliser la fonction native file_get_contents() pour faire un appel GET à un api avec un url.

    <?php

$url = 'https://api.chucknorris.io/jokes/random';

$reponseEnJson = file_get_contents($url);
/*
{
    "categories": [],
    "created_at": "2020-01-05 13:42:29.296379",
    "icon_url": "https://assets.chucknorris.host/img/avatar/chuck-norris.png",
    "id": "bESHJ3YrTEW638ky0rr3Yg",
    "updated_at": "2020-01-05 13:42:29.296379",
    "url": "https://api.chucknorris.io/jokes/bESHJ3YrTEW638ky0rr3Yg",
    "value": "Chuck Norris doesn't need an axe or saw to cut down a tree -- he needs only his foot, or his fist, or his glare."
}
*/
$objetReponse = json_decode($reponseEnJson);

echo $objetReponse->value;
// Chuck Norris doesn't need an axe or saw to cut down a tree -- he needs only his foot, or his fist, or his glare.
?>
